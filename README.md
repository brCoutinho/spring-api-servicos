# API de Serviços Prestados

API Restful para cadastro de Serviços Prestados, onde é possível cadastrar Clientes e Serviços Prestados e atribuir aos serviços o seu respectivo cliente.

```markdown
FUNCIONALIDADES
```

```markdown
-Cadastrar Usuário
	-Usuário
	-Senha
-Logar na Aplicação
	-Usuário
	-Senha
-Cadastrar Clientes
	-Nome
	-CPF
	-Data do Cadastro
-Consultar Clientes
-Deletar Clientes
```
